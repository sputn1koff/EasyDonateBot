Бот уведомлений о покупках EasyDonate в **Telegram** и **VK** + просмотр **последних покупок**.

## Возможности

- Уведомления о новых покупках в Телеграм/Вконтакте
- Статус платежа (успешно / в обработке / отменён`(todo)` / возврат`(todo)`)
- Товары + корректное отображение цены со скидкой
- Логи сервера - `какие команды ушли на сервер для выдачи`
- Кнопка - `Последние покупки`

## Требования

- **Java 17+** (рекомендуется Java 21)
- Доступ к EasyDonate `Ключ Магазина (Shop-Key)`
- Для Telegram: `токен бота`
- Для VK: `токен группы и group-id`

## Как запустить?

- Создаем любую папку на вашей машине - пример: `easydonate-bot`
- Положим нашего `jar-бота` в нашу папку `easydonate-bot`
- далее создаем `screen` и запускаем `бота`
- screen -S easydonate-bot
- cd */easydonate-bot/
- java -jar EasyDonateBot-1.0.jar
